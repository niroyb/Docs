****SSH key****

How to generate a SSH key and add to the Github for enabling a linux client to access HD-VM github project

From a Linux box: 

Here are the steps:

Eg:
root@ubuntu:/home# ssh-keygen -t rsa -b 4096 -C soccerl@microsoft.com
Generating public/private rsa key pair.
Enter file in which to save the key (/root/.ssh/id_rsa):
/root/.ssh/id_rsa already exists.
Overwrite (y/n)? y
Enter passphrase (empty for no passphrase):
Enter same passphrase again:
Your identification has been saved in /root/.ssh/id_rsa.
Your public key has been saved in /root/.ssh/id_rsa.pub.
The key fingerprint is:
SHA256:ZW7yAl55vaWfexHJbW/inaNTaB5IzAO8JxY0X1KeXgk soccerl@microsoft.com
The key's randomart image is:
+---[RSA 4096]----+
|        oo ..E   |
|         +o + o .|
|          O. o.oo|
|         O B. .+o|
|      . S B +.o.o|
|     . o = . B.oo|
|      . . . =.o+o|
|         .   +.+o|
|             .*o.|
+----[SHA256]-----+
root@ubuntu:/home# eval $(ssh-agent -s)
Agent pid 1823
root@ubuntu:/home# ssh-add ~/.ssh/id_rsa
Identity added: /root/.ssh/id_rsa (/root/.ssh/id_rsa)
root@ubuntu:/home# cd /root
root@ubuntu:~# cd .ssh
root@ubuntu:~/.ssh# ls
id_rsa  id_rsa.pub  known_hosts
root@ubuntu:~/.ssh# ls -l
total 12
-rw------- 1 root root 3243 Mar  2 22:11 id_rsa
-rw-r--r-- 1 root root  747 Mar  2 22:11 id_rsa.pub
-rw-r--r-- 1 root root  444 Feb 14 13:52 known_hosts
root@ubuntu:~/.ssh# vi id_rsa
root@ubuntu:~/.ssh# vi id_rsa.pub
root@ubuntu:~/.ssh#

Copy the public key in id_rsa.pub  and add its content as a new SSH key to your GitHub account
https://help.github.com/articles/adding-a-new-ssh-key-to-your-github-account/

cd /home/soccerl

root@ubuntu:/home/soccerl#git clone git@github.com:Microsoft/hd-vm.git hdvm


For Windows client, the operation is similar, you can following the link below
See instructions in https://help.github.com/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent/#platform-windows
